O que é o HTML?

voltar

O HTML não é uma linguagem de programação. Assim como o seu próprio nome define, é uma linguagem de marcação de texto. O HTML pode ser utilizado com diversas linguagens de programação, sendo algumas mais específicas para o Back-End e outras para o Front-End.

A linguagem mais utilizada em conjunto com o HTML é o Javascript. Outra linguagem importante para ser aprendida em conjunto com o HLTM é o CSS, que serve para alterar o layout padrão do HTML

A estrutura básica do HTML é dividida em praticamente 2 partes. O Head em que é definido o título da página e o Body em que é definido o conteúdo da página. O Visual Code

Tags

voltar

A parte mais importante do HTML são as Tags. As Tags são as responsáveis por alterar o formato do texto. Existem diversas Tags e é praticamente impossível saber todas de cabeça, por isso é importante ter o conhecimento do site W3 Schools, pois lá é possível encontrar todas a TAGS presentes no HTML

Durante o curso aprendemos diversar Tags como:

  • h1 até h6 - que define o tamanho especial de texto para títulos e subtítulos
  • p - que define a criação de um parágrafo
  • mark - que faz um marca texto
  • small - que diminui o tamanho da letra
  • i - que modifica o texto para itálico
  • u - sublinha o texto
  • ol - cria uma lista organizada
  • ul - cria uma lista não organizada
  • li - cria o item da lista
  • a - faz os links para outros sites
  • hr - cria uma linha para dividir o título por exemplo
  • sub - faz o texto ficar subscrito
  • blockquote -
    cria um texto pulando linhas
  • del - coloca uma tarja no texto, como se estivesse apagando
  • abbr - define uma abreviatura ou acrônimo
